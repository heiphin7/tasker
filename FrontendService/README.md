# TaskMaster Frontend Service

Фронтенд-сервис для системы управления задачами TaskMaster, который предоставляет пользовательский интерфейс для взаимодействия с другими микросервисами.

## Описание

TaskMaster Frontend Service - это веб-приложение, которое служит единой точкой входа для пользователей системы TaskMaster. Сервис взаимодействует с другими микросервисами (AuthService, UserService, TaskService) через REST API и Kafka, предоставляя пользователям удобный интерфейс для работы с задачами, командами и пользователями.

## Технологии

- Java 17
- Spring Boot 3.4.3
- Spring Security
- Spring WebFlux (для взаимодействия с другими сервисами)
- Thymeleaf (шаблонизатор)
- Tailwind CSS (стили)
- Kafka (для асинхронного взаимодействия)
- Maven (сборка проекта)

## Функциональность

- Аутентификация и авторизация пользователей
- Управление профилем пользователя
- Просмотр и управление задачами
- Работа с подзадачами, комментариями и вложениями
- Управление командами и участниками
- Панель управления с обзором задач и активностей

## Структура проекта

- `src/main/java/com/FrontendService/` - исходный код приложения
  - `config/` - конфигурационные классы
  - `controller/` - контроллеры для обработки HTTP-запросов
  - `model/` - модели данных
  - `service/` - сервисы для взаимодействия с другими микросервисами
  - `util/` - утилитные классы
- `src/main/resources/` - ресурсы приложения
  - `templates/` - Thymeleaf шаблоны
  - `static/` - статические ресурсы (CSS, JavaScript, изображения)
  - `application.properties` - настройки приложения

## Запуск приложения

1. Убедитесь, что у вас установлены Java 17 и Maven
2. Клонируйте репозиторий
3. Перейдите в директорию проекта
4. Выполните команду: `mvn spring-boot:run`
5. Приложение будет доступно по адресу: `http://localhost:8093`

## Взаимодействие с другими сервисами

- **AuthService** - аутентификация и авторизация пользователей
- **UserService** - управление пользователями и командами
- **TaskService** - управление задачами, подзадачами, комментариями и вложениями
